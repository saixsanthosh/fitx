## 🚀 HOW TO BUILD AND RUN

### Step 1: Prerequisites
```bash
# Ensure you have:
- Android Studio Hedgehog or later
- JDK 17
- Android SDK 35
- Gradle 8.0+
```

### Step 2: Open Project
```bash
# Open Android Studio
File → Open → Select fitx folder
```

### Step 3: Sync Gradle
```bash
# Android Studio will automatically sync
# Or manually: File → Sync Project with Gradle Files
```

### Step 4: Build
```bash
# In Android Studio:
Build → Make Project

# Or via command line:
./gradlew assembleDebug
```

### Step 5: Run
```bash
# Connect device or start emulator
# Click Run button (green play icon)
# Or: Run → Run 'app'

# Or via command line:
./gradlew installDebug
```

## 🐛 TROUBLESHOOTING

### Issue: App won't build
**Solution**: 
```bash
# Clean and rebuild
./gradlew clean
./gradlew assembleDebug
```

### Issue: 3D effects not visible
**Solution**: 
- Ensure hardware acceleration is enabled
- Check device supports OpenGL ES 2.0+
- Try on real device (emulator may not show all effects)

### Issue: Haptic feedback not working
**Solution**:
- Check device has vibration motor
- Ensure vibration permission granted
- Check device settings allow vibration

### Issue: Navigation not working
**Solution**:
- Check all screen files are present
- Verify imports are correct
- Rebuild project

### Issue: Theme not changing
**Solution**:
- Check AI personalization is enabled
- Manually select theme in settings
- Restart app
