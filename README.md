## 🚀 Installation

### Prerequisites
- Android Studio Hedgehog or later
- JDK 17
- Android SDK 35
- Gradle 8.0+

### Build from Source

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/fitx.git
cd fitx
```

2. **Open in Android Studio**
```bash
# File → Open → Select fitx folder
```

3. **Sync Gradle**
```bash
# Android Studio will automatically sync
# Or: File → Sync Project with Gradle Files
```

4. **Build**
```bash
# In Android Studio: Build → Make Project
# Or via command line:
./gradlew assembleDebug
```

5. **Run**
```bash
# Connect device or start emulator
# Click Run button (green play icon)
# Or: Run → Run 'app'
```

- Beta releases available on GitHub Releases

## 🏗️ Architecture

### Tech Stack
- **Language**: Kotlin 100%
- **UI Framework**: Jetpack Compose
- **Design**: Material 3
- **Architecture**: Clean Architecture + MVVM
- **DI**: Hilt (Ready)
- **Database**: Room
- **Network**: Retrofit
- **Storage**: DataStore
- **Backend**: Firebase (Optional)
- **Maps**: Google Maps / OSM
- **Charts**: MPAndroidChart
- **Animations**: Lottie, Accompanist

### Project Structure
```
fitx/
├── app/
│   ├── src/main/java/com/fitx/app/
│   │   ├── data/           # Data layer
│   │   ├── domain/         # Domain layer
│   │   ├── ui/             # UI layer
│   │   │   ├── components/ # Reusable components
│   │   │   ├── screens/    # All screens
│   │   │   ├── theme/      # Theme & 3D effects
│   │   │   ├── navigation/ # Navigation system
│   │   │   └── viewmodel/  # ViewModels
│   │   ├── service/        # Background services
│   │   └── util/           # Utilities
│   └── build.gradle.kts
└── docs/                   # Documentation
```
