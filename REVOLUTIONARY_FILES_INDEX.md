## 💡 USAGE EXAMPLES

### Example 1: Add 3D Effect to Any Card
```kotlin
// Import
import com.fitx.app.ui.theme.premium.*

// Use
Parallax3DCard(
    rotationX = 10f,
    rotationY = 10f,
    depth = 30f
) {
    YourCard()
}
```

### Example 2: Add Haptic Feedback
```kotlin
// Import
import com.fitx.app.ui.theme.premium.*

// Use
val hapticEngine = rememberHapticEngine()
Button(onClick = { hapticEngine.playSuccess() }) {
    Text("Click me!")
}
```

### Example 3: Use AI Theme
```kotlin
// Import
import com.fitx.app.ui.theme.premium.*

// Use
val aiTheme = AIThemeEngine()
val theme = aiTheme.getPersonalizedTheme(
    activityLevel = 0.8f,
    moodScore = 0.7f
)
```
